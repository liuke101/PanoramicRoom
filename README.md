# 后端

环境：

jkd1.8 

java8 

springboot2.3.12

MySQL

Maven



# 前端



## 安装项目依赖

```
npm install
```

## 运行项目
```
npm run serve
```

## 打包项目
```
npm run build
```

## 检查和修复文件
```
npm run lint
```

## 停止项目

命令行Ctrl+C
